CSS Selectors

Applicare CSS a parti differenti di HTML

1. L'element selector.

  1. I target del Class selectors si basano sul valore dell'attributo class.
  2. I target dell'id ID selectors si basano sul valore dell'attributo id.
  3. Attribute selectors selezionano elementi basati su attributi e valori.
  4. Il selettore universale coinvolge tutti gli elementi.
